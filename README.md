# Simple Portfolio Site

A personal portfolio website built with React, Vite, and Shadcn UI.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## customization

Update `src/pages/Index.tsx` and other components to customize the content.
